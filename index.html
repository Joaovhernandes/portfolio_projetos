    Cases de Sucesso

    Visualização de Orçamento em Power BI

    Através das técnicas de ETL, foi possível agrupar as contas do planejamento anual realizado por gestores e compará-las as contas realizadas lançadas no sistema SAP.

    A visualização do comparativo 'Real X Orçado' foi disponibilizada em Power BI. Sendo assim, cada etapa do processo foi concebida da seguinte forma:

  • Extract (Extração): Planilhas de orçamento e sistema SAP;
  • Transform (Transformação): Tratamento dos dados e relacionamento de tabelas através do Excel (ferramenta mais adequada para a situação);
  • Load (Carregamento): Publicação do dashboard e liberação para acesso através do serviço do Power BI.

    • Tratamento de dados para concepção de um Radar Chart

    Manipulando os dados através do Python foi possível a classificação e comparação de pontuação em um radar chart (gráfico de radar), onde a pontuação era obtida a partir de respostas de usuários.

    A partir dessa solução foi possível comparar diferentes perfis e classificar quais obteram maior pontuação.
